# Space Cafe PWA

A React PWA cafe ordering prototype.

## Features

- Customer login/register form
- Menu screen
- Cart quantity logic
- Payment simulation
- Customer token generation
- Admin/kitchen token generation
- Order status tracking
- Food collection flow
- Review sharing
- No QR code dependency

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
